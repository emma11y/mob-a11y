# Mob & Accessibilité

Un projet web utilisant Vite comme bundler et dev server.

## Installation

```bash
npm install
```

## Développement

Pour lancer le serveur de développement :

```bash
npm run dev
```

Accédez à http://localhost:5173

## Production

Pour créer la build optimisée :

```bash
npm run build
```

Les fichiers seront générés dans le dossier `dist/`

Pour prévisualiser la build :

```bash
npm run preview
```

## Structure du projet

- `index.html` - Point d'entrée HTML
- `src/main.ts` - Point d'entrée TypeScript
- `style.scss` - Styles SCSS globaux
- `src/assets/` - Ressources (fonts, icônes Font Awesome, etc.)
