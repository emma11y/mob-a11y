# Exercice 1 : Pour commencer doucement : Titres

Vous avez le dossier `tests` qui vous permet de lancer les tests Playwright.

Chaque test correspond à un exercice.

Pour lancer la commande `npx playwright test exercice1`.

Sur chaque exercice, vous devez résoudre les tests pour qu'il passe au vert avant de passer à l'exercice suivante.

Votre mission sur cet exercice est d'attribuer le bon `heading` sur un titre.
