Lecteurs d'écran

Selon votre système d'exploitation, vous pouvez utiliser :

  • NVDA (Windows)
  • VoiceOver (macOS / iOS)
  • Orca (Linux)

Raccourcis de navigation

Utilisez ces touches pour naviguer rapidement avec votre lecteur d'écran :

  • Navigation générale : TAB pour passer d'un élément interactif à l'autre.
  • Titres :
    • NVDA et Orca : H (suivant) ou Shift + H (précédent).
    • VoiceOver : Ctrl + Option + Commande + H
  • Boutons :
    • NVDA et Orca : B (suivant) ou Shift + B (précédent).
    • VoiceOver : Ctrl + Option + Command + L
  • Liens :
    • NVDA et Orca : K (suivant) ou Shift + K (précédent).
    • VoiceOver : Ctrl + Option + Command + L
  • Régions (Landmarks) :
    • NVDA : D (suivante) ou Shift + D (précédente)
    • VoiceOver : Ctrl + Option + U
    • Orca : M (suivante) ou Shift + M (précédente)
  • Liste :
    • NVDA et Orca : L (suivant) ou Shift + L (précédent).
      • Element suivant de la liste : I
      • Element précedent de la liste : Shift + I
    • VoiceOver : Ctrl + Option + Command + X
  • Formulaire :
    • NVDA et Orca : F (suivant) ou Shift + F (précédent).
    • VoiceOver : Ctrl + Option + Command + J
  • Afficher la liste des éléments :
    • NVDA : NVDA + F7 (permet de choisir l'onglet Repères).
    • VoiceOver : CTRL + Option + U (ouvre le rotor pour naviguer entre les repères).
